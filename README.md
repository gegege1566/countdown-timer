# Countdown Timer

オンライン発表時に画面の上に重ねて使える、超ミニマルな常時最前面カウントダウンタイマー。

## 特長

- 常時最前面表示・フレームレス・透過背景対応
- 数字のみのミニマル表示（00:00 + 小さなマークボタン）
- 自由なリサイズ、ウィンドウサイズに合わせて数字も自動拡大／縮小
- 99:99 まで設定可能、数字クリックで編集（▲▼ または直接入力）
- 残り時間に応じた進捗バー（緑→黄→赤）
- 残り1分で背景色が変化、終了で背景点滅＋カウントアップ
- 複数のテーマカラー・フォント、背景／文字色のカスタマイズ
- 背景透過モード

## ショートカット

| キー | 動作 |
|------|------|
| Space | 開始 / 一時停止 |
| R | リセット |
| Esc | アプリ終了 |
| 数字クリック | 時間編集 |

## 操作アイコン

- ✥ 移動（ドラッグ）
- ▶ / ⏸ 開始 / 一時停止
- ↺ リセット
- ⚙ 設定（テーマ・フォント・色・透過）
- ✕ 終了

## ローカル開発

```bash
npm install
npm start
```

## ビルド（Windows）

```bash
npm run build
```

`dist/` にインストーラ (`*.exe`) と portable 版が生成されます。

## GitHub Releases へ自動配布

タグを push すると GitHub Actions が自動でビルド＆Release にアップロードします。

```bash
git tag v1.0.0
git push origin v1.0.0
```
