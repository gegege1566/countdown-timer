# Countdown Timer

オンライン発表・ウェビナーで画面の最前面に重ねて時間管理できる、ミニマルなカウントダウンタイマーです（Windows 専用）。

## ダウンロード

最新版のインストーラーは Releases ページから入手できます。

**👉 [最新リリースをダウンロード](https://github.com/gegege1566/countdown-timer/releases/latest)**

`Countdown.Timer.Setup.x.x.x.exe` をダウンロードし、ダブルクリックで実行するとインストールが始まります（NSIS インストーラー、数秒で完了）。

インストール後、スタートメニューから「Countdown Timer」で起動できます。

> ⚠️ Windows SmartScreen により「不明な発行元」と表示される場合があります。コード署名なしのため正常な挙動です。「詳細情報」→「実行」を選択してください。

## 機能

### タイマー本体
- **常時最前面・フレームレス**のカプセル型ウィンドウ
- **最大 99:59**（99分59秒）まで設定可能
- **自動フォントスケール**: ウィンドウサイズに合わせて数字が拡大／縮小
- 数字部分をドラッグして **自由に移動**、4隅のハンドルで **自由にリサイズ**
- 極小サイズ（60×30px）から大画面まで対応

### 視覚的フィードバック
- **残り1分** で数字色が警告色（オレンジ）に変化
- **超過後** は数字が赤色に + 左に「**+**」マークが表示
- 超過直後の **2.5秒間は数字が点滅**（音は鳴らないため画面で気付ける）
- 超過後もカウントアップを継続表示

### 表示モード
- **黒 / 白** の2テーマをワンクリックで切替
- **背景透過モード**: デスクトップが透けて見える（発表中の邪魔にならない）
- 非透過時は文字色に合わせた **枠線** を自動表示

### 設定の永続化
- ウィンドウサイズ・位置
- 設定時間
- テーマ・背景透過の状態

これらは次回起動時に自動復元されます。

## 操作方法

| アクション | 方法 |
|------------|------|
| 開始 / 一時停止 | ▶ / ⏸ ボタン、または `Space` キー |
| リセット | ↺ ボタン、または `R` キー |
| 時間変更 | 数字を **ダブルクリック** → 直接入力 / ▲▼ / ↑↓ キー → OK |
| ウィンドウ移動 | 数字部分をドラッグ |
| サイズ変更 | ウィンドウ4隅のハンドルをドラッグ |
| テーマ切替（黒↔白） | カプセル下タブの **◐** ボタン |
| 背景透過切替 | カプセル下タブの **▢** ボタン |
| 終了 | カプセル下タブの **✕** ボタン、または `Esc` キー |

カウントダウン中はタブが収納され、カプセル本体と一時停止ボタンだけが表示されます。

## システム要件

- Windows 10 / 11 (64-bit)

## ローカル開発

```bash
git clone https://github.com/gegege1566/countdown-timer.git
cd countdown-timer
npm install
npm start
```

## ビルド

```bash
npm run build
```

`dist/Countdown Timer Setup x.x.x.exe` が生成されます。

リリース手順の詳細は [RELEASING.md](RELEASING.md) を参照してください。

## 永続化データの場所

設定・状態は以下のファイルに保存されています。クリアしたい場合は削除してください。

```
%APPDATA%\countdown-timer\config.json
```
