# Countdown Timer

オンライン発表時に画面の上に重ねて使える、ミニマルな常時最前面カウントダウンタイマー。

## 特長

- 常時最前面・フレームレス・カプセル型ウィンドウ
- 最大 **99:59**（59分59秒）まで設定可能
- 数字の **ダブルクリック** で時間を編集（直接入力 or ▲▼ or 矢印キー）
- ウィンドウをドラッグで移動、**4隅のハンドル** で自由リサイズ
- ウィンドウサイズに合わせて数字フォントが **自動拡大／縮小**
- 残り **1分で数字の色** が変化（警告色）
- 超過後は **数字を赤色** + 左に **+ マーク** + **2.5秒点滅**（背景は変化させない）
- **黒 / 白** の2テーマ切替
- **背景透過モード**（デスクトップを透かして表示）
- 非透過時はテーマの文字色に合わせた枠線を表示
- ウィンドウサイズ・位置・設定時間・テーマ・透過状態を **永続化**（次回起動で復元）

## 操作

| アクション | 方法 |
|------------|------|
| 開始 / 一時停止 | ▶ / ⏸ ボタン、または `Space` |
| リセット | ↺ ボタン、または `R` |
| 時間変更 | 数字をダブルクリック（保存はOKボタン／Enter） |
| 移動 | 数字部分をドラッグ |
| サイズ変更 | 4隅のハンドルをドラッグ |
| テーマ切替 | タブの ◐ ボタン |
| 背景透過切替 | タブの ▢ ボタン |
| 終了 | タブの ✕ ボタン、または `Esc` |

カウントダウン中はタブが引っ込み、カプセルだけが表示されます（一時停止ボタンのみ有効）。

## インストール

最新のインストーラー (`Countdown Timer Setup x.x.x.exe`) を [Releases](https://github.com/gegege1566/countdown-timer/releases) からダウンロードして実行。

## ローカル開発

```bash
npm install
npm start
```

## ビルド

```bash
npm run build
```

`dist/Countdown Timer Setup x.x.x.exe` が生成されます。

## 永続化データ

設定・状態は以下に保存されます：

```
%APPDATA%/countdown-timer/config.json
```
